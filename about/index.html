# About this Handbook

# Background


This Handbook was created to accompany the Toyota Wars board game designed by LtCol. Richard Davis, USAF, and published by Strategy and Tactics in 1991. The original edition of the game ships in a magazine format with everything you need to play, including the game map rulebook, and game pieces.

Unfortunately, the original game can be daunting to learn and play. The rules are complex and explained in dense, technical language. The game map, which was designed and printed in 1991, can be visually confusing by today's standards.

This Handbook is meant to make Toyota Wars more accessible to players. It can be used both to learn about Toyota Wars before purchasing the game, and, once game materials have been received from Strategy and Tactics, it should be used to navigate play rules and consult important charts and resources during gameplay. It is still necessary to purchase the original game from Strategy and Tactics (opens new window).


# Updates included in this Handbook

Game Rules

  • The rulebook now includes additional graphics, explanations, and introductory material for player comprehension. The rules themselves have not been modified.
  • Best of all, because the rules in this Handbook are digitized, you can search them!

Dice

  • The rulebook includes a pair of digital six-sided dice. The Strategy and Tactics magazine does not ship with dice.

Map

  • A redesigned game map is also accessible on this site, and can be printed on US ANSI D paper (22"x34").
  • This Handbook is compatible with both the original map and the redesigned map.

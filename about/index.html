# About this Handbook

# Background


This Handbook was created to accompany the Toyota Wars board game published by Strategy and Tactics in 1991. The original edition of the game ships in a magazine format with everything you need to play, including the game map rulebook, and game pieces.

Unfortunately, editions of the original game are in short supply, and those who do get their hands on a magazine can find the game dauting to learn. The rules are complex and explained in dense, technical language. The game map, which was designed and printed in 1991, can be visually confusing by today's standards.

This Handbook is meant to improve both access to the game and usability of the game rules.


# Updates included in this Handbook

→ Game Rules The rulebook has been edited in order to include additional graphics, explanations, and introductory material for player comprehension. The rules themselves have not been modified. Best of all, because this rulebook is digital, you can now search it!

→ Dice The rulebook includes a pair of digital six-sided dice. The dice do not ship with the original Strategy and Tactics magazine.

→ Map A redesigned game map is also accessible on this site, and can be printed on D size paper (32"x48").

→ Coming soon Game Pieces: Eventually, we hope to redesign the game pieces and link to the image files on this website as well.

With the map, the rules, and the game pieces available with this Handbook, players will be able to play Toyota Wars with the ressources provided on this website alone, without needing to hunt down an ancient copy on Ebay!

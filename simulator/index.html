# Installation Guide

Steam setup

  1. If you don't already have one, register for a Steam account (opens new window) and install the program.
  2. If you don't already have steam friends with whom you intend to play Opération Manta, you may want to invite a friend before launching the game. Log into Steam and on the top menu select Your Username > Friends > Add a Friend. Use any of the various methods to invite.

Game Installation

  1. Log into Steam. In the top menu, select Store. Search for Table Top Simulator. Purchase, and install the simulator. This will provide our game environment.
  2. Select Community > Workshop. Search for "Operation Manta." A workshop item by user HassanDjamous should show up. This is the game mod that will allow us to play the Toyota Wars board game in the simulator.
  3. Select the workshop item and click the green Subscribe button.

Play

  1. Launch Table Top Simulator from your desktop applications folder.
  2. To host a game, select Create Game on the simulator launch screen and click Multiplayer. Name the game and launch the server.
  3. Once the game environment launches, we need to load Toyota Wars. Select Games in the top menu. A game menu will pop up and display workshop subscription. Select Opération Manta.
  4. Once Opération Manta loads, it time to invite the other player! Click the + under your username on the top-right. Select a friend to invite. Alternatively, friends can brows and search active game servers from the launch screen.

Pause

  1. If players want to quit the game and come back to it later, one of them needs to save a local instance that can be hosted at a later time. In the top menu, select Games > Save and Load > Save game.
  2. When picking back up the game, follow the same steps as above, only instead of loading the workshop mod, the player hosting the game will load the local save, which will also be listed in the game menu along with subscriptions.
